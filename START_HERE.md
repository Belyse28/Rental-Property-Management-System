## 🎯 Common Tasks

### Add a Property
1. Login → Click "Manage Properties"
2. Click "+ Add Property"
3. Fill form → Click "Add"

### Add a Tenant
1. Click "Manage Tenants"
2. Click "+ Add Tenant"
3. Select available property
4. Fill details → Click "Add"

### Record Payment
1. Click "Manage Payments"
2. Click "+ Add Payment"
3. Select tenant and property
4. Enter amount → Click "Add"
